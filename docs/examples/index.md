# Examples

This chapter contains runnable example scripts that demonstrate common py-homebox workflows.

Each script is self-contained and loads credentials from a `.env` file in the `examples/` directory.
Copy `examples/.env.sample` to `examples/.env` and fill in your Homebox URL and credentials before running.

| Example | Description |
|---|---|
| [Get Homebox Statistics](get_homebox_statistics.md) | Retrieve and print group, label, location, and time-series statistics |
| [Create Location and Add Items](create_location_add_items.md) | Create a location, add items individually and in bulk, and generate a location label |
| [Item Attachments](item_attachments.md) | Upload, list, and download file attachments on an item |
| [Get Item Labels](get_items_label.md) | Generate printable PNG labels for every item in a location |
| [User Management](user_management.md) | Create a group invitation and register a new user account |
